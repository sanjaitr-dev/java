🔧 One-time Firebase Setup

This app uses Google Firebase (free) for login and cloud-saving your progress. Follow these 4 quick steps:

1
Go to console.firebase.google.com → click Add project (free Spark plan, no credit card needed)
2
In your project → Authentication → Get started → Enable Email/Password sign-in method → Save
3
Firestore Database → Create database → choose Start in test mode → Next → Enable
4
Project Settings (⚙ gear icon) → Your apps → click </> web icon → Register app → copy apiKey and projectId from the config shown

🔒 Your credentials are stored only in this browser. They are sent directly to Firebase's servers and nowhere else.
